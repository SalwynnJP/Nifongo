練習問題１

〔   〕に入る言葉として最も適当なものを、１・２・３・４からひとつ選びなさい。

状況〔   〕、許されるかもしれない。
  1. 最中では
  2. 以上なら
  3. 次第では
  4. 反面なら
高いの安いの言った〔   〕に、何も買わずに帰っていった。
  1. ながら
  2. あげく
  3. ついで
  4. ばかり
年上〔   〕能力があるとはかぎらない。
  1. であればこそ
  2. だからといって
  3. にもかかわらず
  4. ということは
ルールならば〔   〕。
  1. 従うわけない
  2. 従うまでもない
  3. 従うことはない
  4. 従うしかない
最後まで〔   〕じまいだった。
  1. わからない
  2. わからず
  3. わかる
  4. わかった
〔   〕とたんに、雨が降ってきた。
  1. 家を出る
  2. 家を出て
  3. 家を出た
  4. 家を出ない

  1. ないなら
  2. なくして
  3. なければ
  4. なくても

  2. かかわらず
  3. いわせると
  4. したがって
〔   〕からって、逃げるのはよくない。
  1. 逃げよう
  2. 逃げたい
  3. 逃げられない
  4. 逃げてしまう

  2. までに
  3. だけに
  4. ように

毎日記録をするなど、〔   〕この上ない。

  2. 面倒なこと

  4. 面倒にして

  2. については
  3. とすれば
  4. といっても

  2. なら

  4. など
