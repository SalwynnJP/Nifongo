〔   〕に入る言葉として最も適当なものを、１・２・３・４からひとつ選びなさい。

〔   〕。子どもに泣かれると、いつもそう思う。
  1. こっちも泣きたいかのようだ
  2. こっちが泣きたいことはない
  3. こっちが泣きたいくらいだ
  4. こっちが泣きたいに違いにない
よろこびの〔   〕裸(はだか)になった。
  1. もとで
  2. あまり
  3. とおり
  4. くせに
彼は、貧しい〔   〕頭も悪い。
  1. ように
  2. ながら
  3. うえに
  4. ものの
〔   〕最中だから、静かにしてください。
  1. 会議
  2. 会議した
  3. 会議している
  4. 会議する
タバコ〔   〕身体に悪いものもない。
  1. など
  2. だけ
  3. さえ
  4. ほど
大きい〔   〕大きすぎる感じさえする。。
  1. というより
  2. とすれば
  3. といったら
  4. とともに

  1. において
  2. にそって
  3. につれて
  4. にとって

  1. に反し
  2. わりに
  3. くせに
  4. にしろ

  2. たまらず
  3. とわず
  4. かまわず

  2. ながら
  3. なんて
  4. だけに
