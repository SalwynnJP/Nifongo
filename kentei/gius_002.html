語彙（正しい使い方）

練習問題２

次の言葉の使い方として最もよいものを、1・2・3・4からひとつ選びなさい。

執着(しゅうちゃく)
  1. 休み中も健康に執着して生活してください。
  2. お金に対して強い執着を持っているようだ。
  3. テストの前なら、勉強にだけ執着するべきだ。
  4. 世界平和には、各国の平和への執着が必要だ。
さかり
  1. 2011年の大会をさかりに成績がさがってきた。
  2. 北海道では、多くの地域で観光業がさかりだ。
  3. エネルギーをめぐる議論がさかりに行われている。
  4. 花はさかりを過ぎていたが、十分に美しかった。
サイレン
  1. 危険を知らせるサイレンが鳴った。
  2. 10年のサイレンで見直す計画だ。
  3. 現代ではサイレンが発達している。
  4. 申請書類(しんせいしょるい)にはサイレンが必要です。
こっそり
  1. リンゴが安いので、重いがこっそり買ってしまった。
  2. 村では人々が自然に囲まれこっそりと暮らしている。
  3. お店の人が私にだけこっそり作り方を教えてくれた。
  4. 夜の公園はこっそりとして、虫の声だけが聞こえた。
はやる
  1. 駅前は平日でも多くの観光客ではやっている。
  2. 風邪がはやっているので気をつけてください。
  3. 韓国の芸能人でだれが最もはやっていますか。
  4. 暑いので最近はアイスがよくはやっています。
ぴかぴか
  1. 兄弟２人で部屋をぴかぴかに掃除した。
  2. 門の前では火がぴかぴかと燃えている。
  3. 池に映った月がぴかぴかと揺れている。
  4. 暑さで練習の後にはぴかぴかになった。
余裕(よゆう)
  1. 土曜は予約でいっぱいですが、金曜はまだ余裕があります。
  2. 余裕の人員を整理するため、100人以上の社員を解雇した。
  3. お腹についた余裕な脂をとるため、毎日運動をしている。
  4. 10人分余裕につくっておいたが、それでも足りなかった。
わざと
  1. たった書類１枚のためにわざと家に帰るのは面倒だ。
  2. 「人の前でわざと悪いことを言う必要ないだろう。」
  3. 遠いところからわざと病気のお見舞いにきてくれた。
  4. わざと買ってきたのですから、ぜび使ってください。
ためす
  1. 電波を使って物の間の距離(きょり)をためす。
  2. 彼女は食べ物には興味(きょうみ)をためさない。
  3. 性能をためすために実験をしている。
  4. 警察官が事故の原因をためしている。
しだいに
  1. 道路の幅がしだいにせまくなっていく。
  2. ５時間かかってしだいに頂上についた。
  3. しだいに目的地に到着するころだろう。
  4. しだいに会っていないが、元気そうだ。
