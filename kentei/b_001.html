練習問題１

〔   〕に入る言葉として最も適当なものを、１・２・３・４からひとつ選びなさい。

天気予報〔   〕、今週は晴れる日が多いらしい。
  1. によって
  2. によれば
  3. により
  4. によった
仕事が山の〔   〕残っている。
  1. らしく
  2. ほどに
  3. ように
  4. ままに
子どものころは、川でよく泳いだ〔   〕。
  1. までだ
  2. ことだ
  3. ままだ
  4. ものだ
大会では、学校の代表〔   〕頑張(がんば)るつもりです。
  1. にせよ
  2. として
  3. というと
  4. にしては
あまり勉強していなかったのに、２級に合格した。この結果には、自分のこと〔   〕驚(おどろ)いている。
  1. なんて
  2. だから
  3. ながら
  4. になり
今年のカキは〔   〕味も良い。
  1. 大きくて
  2. 大きく
  3. 大きさや
  4. 大きいとか
毎回出席〔   〕単位がとれなかった。
  1. したのに
  2. したから
  3. するけど
  4. するが
あなたが病気になったのは、不規則な生活の〔   〕ほかならない。
  1. 結果で
  2. 結果に
  3. 結果なら
  4. 結果には
こんな大切なときに、カゼ〔   〕寝ていられるか。
  1. ぬきには
  2. なんかで
  3. ばかりに
  4. ながらに
今年の大会は中止しますが、来年以降も開かないという〔   〕。
  1. どころではありません
  2. こととなっています
  3. わけではありません
  4. にほかなりません
