練習問題９

〔   〕に入る言葉として最も適当なものを、１・２・３・４からひとつ選びなさい。

死んだ人が生き返るなんて、事実に〔   〕作られたとは思えない。
  1. ともなって
  2. もとづいて
  3. かかわらず
  4. くらべて
全体を〔   〕特に間違っているところはない。
  1. めぐり
  2. はじめ
  3. 通じて
  4. 問わず
案内に〔   〕進んでください。
  1. こたえて
  2. もとづいて
  3. ともなって
  4. したがって
事前に心配した〔   〕、特に何もなく終わった。
  1. とおりに
  2. だけあって
  3. どころか
  4. わりには
経験〔   〕やる気のある人を求めます。
  1. のみならず
  2. を問わず
  3. に限らず
  4. とは言わず
この結果〔   〕練習が足りなかったということだろう。
  1. によっては
  2. といっても
  3. だけあって
  4. からすると
景気は、回復するどころか〔   〕。
  1. すでに回復している
  2. 次第に回復しつつある
  3. さらに悪くなっている
  4. 悪くないようがない
テレビを〔   〕勉強してはいけない。
  1. 見るにあたり
  2. 見ながら
  3. 見るにつれ
  4. 見る反面
悪くならない〔   〕医者に行くべきだ。
  1. わりに
  2. だけに
  3. うちに
  4. くせに
家を建てる〔   〕しばらく先だろう。
  1. と言えば
  2. にしても
  3. ことには
  4. ものだから
