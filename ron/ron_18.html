テーマの決め方

テーマをどのように選(えら)ぶか

報告(ほうこく)では、テーマがあらかじめ与(あた)えられていることが多(おお)い。他方(たほう)、小論文(しょうろんぶん)では、基本的(きほんてき)にテーマを自分(じぶん)で決(き)めなければならない。

テーマ決定(けってい)の手順(てじゅん)

テーマを決(き)めるときには、次(つぎ)のように考(かんが)えるとよい。

  1. 大(おお)きなテーマを決(き)める
  2. 小(ちい)さなテーマに絞(しぼ)る
  3. 具体的(ぐたいてき)な問題(もんだい)を見(み)つける

テーマは、基本的(きほんてき)に、自分自身(じぶんしじん)の興味(きょうみ)や関心(かんしん)に従(したが)って決(き)めていけばよいものである。ただし、内容(ないよう)が新(あたら)しいテーマ（特(とく)に、アニメやゲームなどのサブカルチャーについて）を選(えら)んだときには、参考(さんこう)にできる資料(しりょう)が少(すく)ないなどの問題(もんだい)が生(しょう)じる可能性(かのうせい)があるので注意(ちゅうい)が必要(ひつよう)である。

一方(いっぽう)、「具体的(ぐたいてき)な問題(もんだい)」には、論文(ろんぶん)にふさわしいものと、論文(ろんぶん)にはふさわしくないものとがある。具体的(ぐたいてき)な問題(もんだい)を見(み)つける作業(さぎょう)では、自分(じぶん)の考(かんが)えた問題(もんだい)が論文(ろんぶん)にふさわしいものかどうかを見極(みきわ)めなければならない。

具体的(ぐたいてき)な問題(もんだい)の発見(はっけん)

ここでは〈具体的(ぐたいてき)な問題(もんだい)を見(み)つける〉際(さい)の注意点(ちゅういてん)を述(の)べる。

論文(ろんぶん)にふさわしくない問題(もんだい)は次(つぎ)のようなものである。

  1. すでに答(こた)えがわかっている
    • 「砂糖(さとう)と塩(しお)はどちらが甘(あま)いか？」 → 砂糖(さとう)が甘(あま)い
    • 「神(かみ)は存在(そんざい)するか？」 → 存在(そんざい)しない
  2. 答(こた)えがない問題(もんだい)である
    • 「人間(にんげん)はどう生(い)きるべきか？」 → 特定(とくてい)の答(こた)えがあるとは思(おも)えない
    • 「なぜ神(かみ)は存在(そんざい)するか？」 → 問題(もんだい)が間違っている（神(かみ)は存在(そんざい)しないので、神(かみ)が存在(そんざい)する理由(りゆう)などない）
  3. 手掛(てが)かりがない問題(もんだい)である
    • 「100年後(ねんご)の育達(いくたつ)はどうなっているか？」 → 予測不可能(よそくふかのう)である
    • 「なぜ『さくら』は『さくら』というのか？」 → 証拠(しょうこ)が残(のこ)っていないので答(こた)えが出(だ)せない
  4. 問題(もんだい)のスケールが大(おお)きすぎる
    • 「どうすれば環境問題(かんきょうもんだい)は解決(かいけつ)するか？」 → 問題(もんだい)が大(おお)きすぎ、ひとつの論文(ろんぶん)で答(こた)えを出(だ)すのは無理(むり)だろう
    • 「人生(じんせい)とは何(なに)か？」 → 問題(もんだい)が抽象的(ちゅうしょうてき)すぎる
  5. 問題(もんだい)にする必然性(ひつぜんせい)がない
    • 「明日(あす)の夕食(ゆうしょく)は何(なに)にすべきか？」 → 何(なに)を食(た)べようが個人(こじん)の自由(じゆう)だ
    • 「なぜ私(わたし)は王(おう)さんが好(す)きか？」 → 問題(もんだい)が個人的(こじんてき)すぎる
  6. 問題(もんだい)にする状況(じょうきょう)が変化(へんか)しやすい
    • 「誰(だれ)が首相(しゅしょう)にふさわしいか？」 → 政治状況(せいじじょうきょう)が変(か)われば、比較(ひかく)の対象(たいしょう)も変(か)わってしまう
    • 「なぜ今日(きょう)は晴(は)れているのか？」 → 「今日(きょう)」はすぐ「昨日(きのう)」になってしまう...

〈具体的(ぐたいてき)な問題(もんだい)を見(み)つける〉ときには、自明(じめい)ではないが答(こた)えが出(だ)せる問題(もんだい)を考(かんがえ)え、大(おお)きすぎないものを選(えら)ぶようにするとよい。

テーマの選択理由(せんたくりゆう)

テーマを決(き)めるときには、『何(なに)をテーマにするか』だけでなく、『なぜそのテーマにするのか』も考(かんがえ)えなければならない。

そもそも、テーマは、論(ろんじる)じる価値(かち)のあるものでなければならない。『なぜそのテーマにするのか』については、次(つぎ)の２点(てん)からまとめるとわかりやすい。

  1. テーマへの興味(きょうみ)（なぜそのテーマに関心(かんしん)があるのか）
    1. 理論的(りろんてき)な興味(きょうみ)・関心(かんしん)（なぜそのテーマに答(こた)えを与(あた)える必要(ひつよう)があるのか）
    2. 自分自身(じぶんじしん)の経験(けいけん)・体験(たいけん)（何(なに)がテーマの決定(けってい)に影響(えいきょう)を与(あた)えたのか）
  2. テーマの重要性(じゅうようせい)（そのテーマを研究(けんきゅう)することにどのような意義(いぎ)があるのか）
