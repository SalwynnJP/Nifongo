練習問題

やりもらい

練習１

 次(つぎ)のそれぞれの文(ぶん)について、文法的(ぶんぽうてき)に正(ただ)しい場合(ばあい)は○をつけ、正(ただ)しくない場合(ばあい)は正(ただ)しい文(ぶん)に書(か)き換(か)えなさい。

 答え ←マウスオーバーで解答例を表示

  1. ジェフさんは私からおみやげをもらいました。
     → 私はジェフさんにおみやげをあげました。
  2. 楊さんのお父さんが写真をくれました。
     → ○（楊さんのお父さんが写真をくれました。）
  3. 先生にブランデーをさしあげた。
     → ○（先生にブランデーをさしあげた。）
  4. 陳くんが私にＣＤをあげました。
     → 陳くんが私にＣＤをくれました。
  5. 社長さんが社員に和菓子をくださった。
     → 社長さんが社員に和菓子をあげた。
  6. 呂さんが弟にクッキーをくれました。
     → ○（呂さんが弟にクッキーをくれました。）
  7. アンジェラさんはお父さんにDVDをいただいた。
     → アンジェラさんはお父さんにDVDをもらった。
  8. ペットのサルにエサをやった。
     → ○（ペットのサルにエサをやった。）
  9. これは友人がくれたものです。
     → ○（これは友人がくれたものです。）
  10. 黄さんが私にマンガあげました。
     → 黄さんが（私に）マンガをくれました。
     → （私は）黄さんにマンガをもらいました。
  11. ヘンリーさんはキャサリンさんに写真をとってもらった。
     → ○（ヘンリーさんはキャサリンさんに写真をとってもらった。）
  12. 林さんは私にラーメンを作ってもらいました。
     → 私は林さんにラーメンを作ってあげた。
  13. 先生に文法を教えてくださいました。
     → 先生に文法を教えていただきました。
     → 先生が文法を教えてくださいました。
  14. ちょっと待っていただきますか。
     → ちょっと待っていただけますか。
     → ちょっと待ってくださいますか。
  15. パクさんがおすしをごちそうしてくれました。
     → ○（パクさんがおすしをごちそうしてくれました。）
  16. 必ず試験にパスしてあげる。
     → 必ず試験にパスしてやる。
  17. 天気が晴れてくださった。
     → 天気が晴れてくれた。
  18. 台北の街を案内してさしあげる。
     → ○（台北の街を案内してさしあげる。）
  19. 荷物を取ってきてくれ。
     → ○（荷物を取ってきてくれ。）
  20. 仕事を休ませてもらえませんか。
     → ○（仕事を休ませてもらえませんか。）

練習２

次(つぎ)のそれぞれの文(ぶん)を、同(おな)じ内容(ないよう)を表(あら)わす別(べつ)の授受表現(じゅじゅひょうげん)に書(か)き換(か)えなさい。

 答え ←マウスオーバーで解答例を表示

  1. 山田さんにジュースをもらった。
     → 山田さんがジュースをくれた。
  2. 先生が弟に参考書をくださった。
     → 弟が先生に参考書をいただいた。
  3. 祖父がおこづかいをくれた。
     → 祖父におこづかいをもらった。
  4. キムさんは沢田さんにキムチをあげた。
     → 沢田さんはキムさんにキムチをもらった。
  5. 部長にネクタイをいただいた。
     → 部長がネクタイをくださった。
  6. 部長に飲みに連れていっていただいた。
     → 部長が飲みに連れていってくださった。
  7. ちょっと手伝ってもらえませんか。
     → ちょっと手伝ってくれませんか。
  8. 先生が文法を親切に教えてくださった。
     → 先生に文法を親切に教えていただいた。
  9. アンドリューさんに英語を教えてもらった。
     → アンドリューさんが英語を教えてくれた。
  10. 兄がジュースを買ってくれた。
     → 兄にジュースを買ってもらった。
