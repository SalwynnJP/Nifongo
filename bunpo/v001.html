練習問題

ヴォイス：受動態

練習

 能動態(のうどうたい)の文(ぶん)は受動態(じゅどうたい)の文(ぶん)に、受動態(じゅどうたい)の文(ぶん)は能動態(じゅどうたい)の文(ぶん)に書(か)き換(か)えなさい。

例：「弟が兄を殴る。」（能動態の文）
 → （受動態の文）「兄が弟に殴られる。」
例：「兄が弟に殴られる。」（受動態の文）
 → （能動態の文）「弟が兄を殴る。」

 答え ←マウスオーバーで解答例を表示

  1. 私は夜中に人に来られた。
     → 夜中に人が来た。
  2. 投手が打者の頭にボールをぶつける。
     → 打者は投手に頭にボールをぶつけられた。
  3. 年老いた親は子に頼る。
     → 子は年老いた親に頼られる。
  4. 子どもが泣いた。
     → 子供に泣かれた。
  5. 誰かに足を踏まれた。
     → 誰かが私の足を踏んだ。
  6. マーくんは高級な魚を釣った。
     → 高級な魚はマーくんに釣られた。
  7. 背の高い人が前に立った。
     → 背の高い人に前に立たれた。
  8. レンブラントは、この絵を描いた。
     → この絵は、レンブラントによって描かれた。
  9. 私の会社は事務員にお金を盗まれた。
     → 事務員が私の会社のお金を盗んだ。
  10. エックス線はレントゲンによって発見された。
     → レントゲンがエックス線を発見した。
  11. 息子が父の頭をバットで殴った。
     → 父は息子に頭をバットで殴られた。
  12. 先生が宿題をたくさん出した。
     → （私たちは）先生に宿題をたくさん出された。
  13. 専務が社長の花瓶を割った。
     → 社長は専務に花瓶を割られた。
  14. 友だちにケーキを食べられた。
     → 友だちが私のケーキを食べた。
  15. 中国人にカバンを盗まれた。
     → 中国人が私のカバンを盗んだ。
