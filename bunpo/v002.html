練習問題

ヴォイス：可能態

練習１

 能動態(のうどうたい)の文(ぶん)は可能態(かのうたい)の文(ぶん)（可能表現(かのうひょうげん)）に、可能態(かのうたい)の文(ぶん)（可能表現(かのうひょうげん)）は能動態(じゅどうたい)の文(ぶん)に書(か)き換(か)えなさい。

例：「Ｑさんが中国語を話す。」（能動態の文）
 → （可能態の文）「Ｑさんには中国語が話せる。」
例：「Ｑさん(に)は中国語が話せる。」（可能態の文）
 → （能動態の文）「Ｑさんが中国語を話す。」

 答え ←マウスオーバーで解答例を表示

  1. 王くんが台湾料理を作る。
     → 王くん(に)は台湾料理が作れる。
  2. 兄が車を運転する。
     → 兄(に)は車が運転できる。
  3. 弟にも車の運転ができる。
     → 弟も車の運転をする。
  4. スミスさんがマラソンを走る。
     → スミスさん(に)はマラソンが走れる。
  5. 宿題の作文が書けた。
     → 宿題の作文を書いた。

練習２

 それぞれの文(ぶん)を、同(おな)じ意味(いみ)を表(あら)わす別(べつ)の文(ぶん)に書(か)き換(か)えなさい。

例：「Ｑさんには中国語が話せる。」（可能表現）
 → （別の可能表現）「Ｑさんは中国語を話すことができる。」
例：「Ｑさんは中国語を話すことができる。」（可能表現）
 → （別の可能表現）「Ｑさんには中国語が話せる。」

 答え ←マウスオーバーで解答例を表示

  1. 彼女は難しい暗算をすることができる。
     → 彼女には難しい暗算ができる。
  2. ペーさんは誕生日が覚えられる。
     → ペーさんは誕生を覚えることができる。
  3. アブドラさんは、おいしい和食がつくれる。
     → アブドラさんは、おいしい和食をつくることができる。
  4. ここの水は飲める。
     → ここの水は飲むことができる。
  5. あと１時間で着くことができる。
     → あと１時間で着ける。

練習３

 それぞれの文(ぶん)について、受動態(じゅどうたい)の文(ぶん)か可能態(かのうたい)の文(ぶん)か自発態(じはつたい)の文(ぶん)か答(こた)えなさい。

 答え ←マウスオーバーで正解を表示

  1. ウサギがヘビに食べられる。
     → 受動態
  2. この店では、おいしい料理が食べられる。
     → 可能態
  3. このパッケージは手で切れる。
     → 可能態
  4. この服は、屋外でも着られる。
     → 可能態
  5. パジャマをひとりで着られる。
     → 可能態
  6. Ｔシャツをナイフで切られる。
     → 受動態
  7. 昔のことが懐かしく思われる。
     → 自発態
  8. 好きな人に思われる。
     → 受動態
  9. 毎朝８時に起きられる。
     → 可能態
  10. 毎朝８時に起こされる。
     → 受動態
