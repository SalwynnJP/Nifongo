社内文書と社外文書

【参考→ビジネス文書】

練習

 次(つぎ)の文書(ぶんしょ)は《社内文書(しゃないぶんしょ)》と《社外文書(しゃがいぶんしょ)》のどちらですか？

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 取引先に製品の部品を注文する文書 → 社外文書
  2. 出張の内容を報告する文書 → 社内文書
  3. 得意先に差し出す年賀はがき（年賀状） → 社外文書
  4. 社内運動会の日程を案内する手紙 → 社内文書
  5. 取引先の工場の見学を依頼する文書 → 社外文書
  6. 海外企業への注文をキャンセルする文書 → 社外文書
  7. セミナーの実施を営業部員に連絡する文書 → 社内文書
  8. 商品の配送ミスを購入者に詫びる手紙（詫び状） → 社外文書
  9. 営業会議の議事録 → 社内文書
  10. 得意先に社名の変更を通知する文書 → 社外文書
  11. それぞれの部署にOA機器の使用状況の調査を依頼する文書 → 社内文書
  12. 転勤の挨拶状 → 社外文書
  13. 病気で会社を休むことを届け出る文書（欠勤届） → 社内文書
  14. 代金の支払いを督促する文書（督促状） → 社外文書
  15. 納期遅れへの苦情を伝える文書 → 社外文書
  16. 納期遅れを取引先に詫びる文書（詫び状） → 社外文書
  17. 工場見学のお礼を述べる手紙（礼状） → 社外文書
  18. 個人情報の取扱規定 → 社内文書
  19. 就業規則 → 社内文書
  20. 業務日報 → 社内文書
