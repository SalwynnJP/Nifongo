通知文書(つうちぶんしょ)とは？

通知文書(つうちぶんしょ)とは、関係者(かんけいしゃ)に必要(ひつよう)な事項(じこう)を知(し)らせるために書(か)く社内文書(しゃないぶんしょ)のことである。

通知文書(つうちぶんしょ)の種類(しゅるい)

会議(かいぎ)の開催(かいさい)や研修会(けんしゅうかい)の実施(じっし)などを参加対象者(さんかたいしょうしゃ)に知(し)らせる文書(ぶんしょ)、希望者(きぼうしゃ)や意見(いけん)を募集(ぼしゅう)する文書(ぶんしょ)、停電(ていでん)やシステムメンテナンスなどを知(し)らせる文書(ぶんしょ)などがある。

通知文書(つうちぶんしょ)の基本(きほん)

通知文書(つうちぶんしょ)は、情報(じょうほう)を知(し)らせることを目的(もくてき)とする社内文書(しゃないぶんしょ)なので、必要(ひつよう)な情報(じょうほう)を明確(めいかく)にわかりやすく書(か)くことが重要(じゅうよう)である。そのために、以下(いか)のポイントに注意(ちゅうい)して書(か)くようにする。

何(なに)に関(かん)する通知(つうち)なのか、誰(だれ)が何(なに)をすればよいのかをはっきりと書(か)く

問(と)い合(あ)わせの必要(ひつよう)がないように、必要(ひつよう)な情報(じょうほう)を漏(も)れなく記載(きさい)する

敬語表現(けいごひょうげん)は最小限(さいしょうげん)のものにする（あまりていねいに書(か)く必要(ひつよう)はない）

あいさつなどの儀礼的(ぎれいてき)な表現(ひょうげん)はできるだけ省略(しょうりゃく)する

通知文書(つうちぶんしょ)の書(か)き方(かた)

例１：会議(かいぎ)の開催通知(かいさいつうち)

営発第28号
平成21年４月７日
営業部員各位
営業部長

営業会議の開催について（通知）

下記のとおり、営業会議を行なうので、出席してください。

記
  1. 日時   ４月20日（水）10時から12時まで
  2. 場所   ミーティングルーム（４階）
  3. 議題   次期営業計画について
  4. 添付   前期売上表
  5. 備考   上記議題に関係する資料を持参のこと
以上

通知文書(つうちぶんしょ)では、あいさつ文(ぶん)や敬語(けいご)は不要(ふよう)である（「×開催いたしますので、」、「×万障お繰り合わせの上、ご出席くださいますようお願い申し上げます。」などとはしない）

指示(しじ)を伝(つた)える文書(ぶんしょ)なので、一般的(いっぱんてき)には「行ないますので」のようにはしない（「行なうので」「開催するので」のようにする）

必要(ひつよう)な情報(じょうほう)は、記書(きが)き（箇条書(かじょうが)き）にしてわかりやすくまとめる

日時(にちじ)・場所(ばしょ)だけでなく、議題(ぎだい)や必要(ひつよう)な書類(しょるい)についても書(か)いた方(ほう)がよい

表現(ひょうげん)のバリエーション

下記(かき)のとおり、営業会議(えいぎょうかいぎ)を行(おこ)なうので、出席(しゅっせき)してください。
→ 下記(かき)の通(とお)り、販売会議(はんばいかいぎ)を開催(かいさい)するので、ご出席(しゅっせき)ください。
［※会議参加者(かいぎさんかしゃ)が、支店長(してんちょう)やマネージャーなどの場合(ばあい)には、ややていねいな表現(ひょうげん)を使(つか)うことがある］
→ 企画課(きかくか)ミーティングを下記(かき)により開催(かいさい)するので、出席(しゅっせき)願(ねが)います。
［※通知文書(つうちぶんしょ)で伝(つた)えられる内容(ないよう)は、一般(いっぱん)に決定事項(けっていじこう)（＝指示(しじ)・命令(めいれい)）である。「願(ねが)います」とした場合(ばあい)には、通知文書(つうち)よりも拘束力(こうそくりょく)の弱(よわ)い依頼文書(いらいぶんしょ)や案内文書(あんないぶんしょ)と思(おも)われる可能性(かのうせい)があるので注意(ちゅうい)すること］
議題(ぎだい)   次期営業計画(じきえいぎょうけいかく)について
→ 議題(ぎだい)   別紙参照(べっしさんしょう)
［※議題(ぎだい)が多(おお)いときや複雑(ふくざつ)なときには、別紙(べっし)にまとめるとよい］
備考(びこう)   上記議題(じょうききだい)に関係(かんけい)する資料(しりょう)を持参(じさん)のこと
→ 資料(しりょう)   当日(とうじつ)席上(せきじょう)で配布(はいふ)
→ 備考(びこう)   前回(ぜんかい)配布済(はいふず)みの資料(しりょう)（「次期中期計画概要(じきちゅうきけいかくがいよう)」）を持参(じさん)すること
［※持ってくるべき資料(しりょう)があるときは、はっきりと指示(しじ)する］
→ 備考(びこう)   添付資料(てんぷしりょう)に記入(きにゅう)のうえ前日(ぜんじつ)までに提出(ていしゅつ)のこと
［※提出(ていしゅつ)するものがあるときは、締切(しめきり)をはっきりと指示(しじ)する］
→ ※なお、○○を○月○日までに○○に提出(ていしゅつ)してください。
［※副文(ふくぶん)として箇条書(かじょうが)きの項目(こうもく)の下部(かぶ)（「以上(いじょう)」の上(うえ)）に書(か)き添(そ)えるときは、「なお、」「その他(ほか)、」などの語句(ごく)をつけて、主文(しゅぶん)と同様(どうよう)の表現(ひょうげん)を用(もち)いる］
  • ミーティングルーム【名詞】 (meeting room)［※会議室(かいぎしつ)の意味(いみ)］
  • ミーティング【名詞】 (meeting)

募集要項(ぼしゅうようこう)に書(か)かれていても、申込方法(もうしこみほうほう)を明記(めいき)しておくようにする

細部(さいぶ)の問合(といあわ)せが予想(よそう)されるので、問合(といあわ)せ先(さき)（担当者(たんとうしゃ)）を書(か)いておく（箇条書(かじょうが)きの項目(こうもく)に含(ふく)めるのもよい）

標記(ひょうき)について、……
→ このたび、エコビジネスの展開(てんかい)を図(はか)るため、環境(かんきょう)プロジェクトチームを立(た)ち上(あ)げることとなりました。ついては、……
［※プロジェクト発足(ほっそく)の経緯(けいい)や目的(もくてき)などを簡単(かんたん)に説明(せつめい)してもよい］
課員(かいん)に周知(しゅうち)してください。
→ 所属(しょぞく)の社員(しゃいん)に周知(しゅうち)させてください。
［※「（課長(かちょう)が課員(かいんに)に）周知(しゅうち)してください。」と「（課長(かちょう)が課員(かいんに)に）周知(しゅうち)させてください。」は、同(おな)じ意味(いみ)になる］
  • プロジェクトメンバー【名詞】 project member
  • 研究協力者【名詞】（けんきゅうきょうりょくしゃ）
  • 周知する【３類動詞】（しゅうちする）
  • 募集要項【名詞】（ぼしゅうようこう）
  • エコビジネス【名詞】 (eco-business)［※「環境(かんきょう)ビジネス」ともいう］

［※重大(じゅうだい)な問題(もんだい)が生(しょう)じる可能性(かのうせい)があるときには、対処方法(たいしょほうほう)を明確(めいかく)に指示(しじ)しておく］

    ［※「セキュリティー」とも書(か)くが、現在(げんざい)では「セキュリティ」の方(ほう)が一般的(いっぱんてき)］
  • セキュリティ強化【名詞】（セキュリティきょうか）
  • 施工者【名詞】（しこうしゃ／せこうしゃ）

通知(つうち)メールの書(か)き方(かた)

比較的(ひかくてき)簡単(かんたん)な連絡事項(れんらくじこう)やコンピュータシステムなどに関(かか)わる事項(じこう)では、電子(でんし)メールを使(つか)うことも多(おお)い。

電子(でんし)メールでは、紙(かみ)の社内文書(しゃないぶんしょ)とは書(か)き方(かた)が異(こと)なる点(てん)もある。以下(いか)では、通知(つうち)メールの書(か)き方(かた)について説明(せつめい)する。

例１：会議(かいぎ)の招集通知(しょうしゅうつうち)メール

作成：販売促進会議のお知らせ
 
鈴木 洋之 <m_suzuki@xxx.co.jp>
mr_list@xxx.co.jp
販売促進会議のお知らせ

各位

総務部総務課 鈴木です。

下記の通り販売促進会議を行ないますので、お集まりください。

都合により参加できない方は、あらかじめご連絡ください。


●日時：９月23日

●場所：５階会議室

●議題：(1)上半期の販売実績報告、(2)下半期の販促計画について

●出席者：総務部長、販売課長、マネージャー各位、販売本部各位


以上、よろしくお願いします。

本文(ほんぶん)は『です・ます体(たい)』で平易(へいい)に書(か)けばよい（社内文書特有(しゃないぶんしょとくゆう)の言(い)い回(まわ)しなどは、特(とく)に使(つか)わなくてもかまわない）

文書番号(ぶんしょばんごう)や発信者名(はっしんしゃめい)、標題(ひょうだい)などは特(とく)に書(か)かなくてもかまわない

箇条書(かじょうが)きにするときも、「記(き)」「以上(いじょう)」などは省略(しょうりゃく)して簡潔(かんけつ)にする

メールでは、すべて左寄せにしてかまわない（逆(ぎゃく)に見(み)づらくなるので、スペースなどを打(う)ち込(こ)んで無理(むり)に位置(いち)を揃(そろ)えないこと）

表現(ひょうげん)のバリエーション

下記(かき)の通(とお)り販売促進会議(はんばいそくしんかいぎ)を行(おこ)ないますので、お集(あつ)まりください。
→ 営業計画(えいぎょうけいかく)に関(かん)する会議(かいぎ)を行(おこ)ないますので、関係者(かんけいしゃ)は必(かなら)ずご出席(しゅっせき)ください。
都合(つごう)により参加(さんか)できない方(かた)は、あらかじめご連絡(れんらく)ください。
→ 欠席(けっせき)の場合(ばあい)は、本(ほん)メールの返信(へんしん)にてお知(し)らせください。
  • 販売促進【名詞】（はんばいそくしん）［※「販促(はんそく)」と略(りゃく)すことも多(おお)い］
  • あらかじめ［予め］【副詞】（あらかじめ）
  • 販売実績【名詞】（はんばいじっせき）
  • 販促計画【名詞】（はんそくけいかく）
  • マネージャー【名詞】(manager)
