問題１

 社内文書(しゃないぶんしょ)では、ａ〜ｉは、書類(しょるい)のどの場所(ばしょ)に書(か)きますか。図(ず)の１〜９の番号(ばんごう)で答(こた)えてください。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

ａ：受信者名 → ３ｂ：記書き → ７ｃ：主文 → ６
ｄ：文書番号 → １ｅ：担当者名 → ９ｆ：件名（標題） → ５
ｇ：結語 → ８ｈ：発信者名 → ４ｉ：発信年月日 → ２

問題２

 社内文書(しゃないぶんしょ)では、次(つぎ)の図(ず)の①〜⑨の場所(ばしょ)に何(なに)を書(か)きますか。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

１ → 文書番号２ → 発信年月日３ → 受信者名
４ → 発信者名５ → 件名（標題）６ → 主文（本文）
７ → 記書き（別記）８ → 結語９ → 担当者名・内線番号
