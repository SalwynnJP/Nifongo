練習

問題１

 社外文書(しゃがいぶんしょ)では、ａ〜ｉは、書類(しょるい)のどの場所(ばしょ)に書(か)きますか。図(ず)の１〜13の番号(ばんごう)で答(こた)えてください。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

ａ：受信者名 → ３ｂ：記書き → 10ｃ：主文 → ７
ｄ：文書番号 → １ｅ：担当者名 → 13ｆ：末文 → ８
ｇ：同封書類 → 11ｈ：発信者名 → ４ｉ：発信年月日 → ２
ｊ：前文 → ６ｋ：記書きの結語 → 12ｌ：件名（標題） → ５

問題２

 社外文書(しゃがいぶんしょ)では、次(つぎ)の図(ず)の①〜⑨の場所(ばしょ)に何(なに)を書(か)きますか。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

１ → 文書番号２ → 発信年月日３ → 受信者名
４ → 発信者名５ → 件名（標題）６ → 前文（本文）
７ → 主文８ → 末文９ → 結語
10 → 記書き（別記）11 → 同封書類12 → 記書きの結語
13 → 担当者名・電話番号
