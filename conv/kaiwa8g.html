練習問題

「〜てみる」

練習１

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：イチゴのケーキ・作る
 → イチゴのケーキを作ってみます。
 → イチゴのケーキを作ってみてください。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 問題(もんだい)・解(と)く
     → 問題を解いてみます。
     → 問題を解いてみてください。
  2. 会社(かいしゃ)・行く
     → 会社に行ってみます。
     → 会社に行ってみてください。
  3. ともだち・聞(き)く
     → ともだちに聞いてみます。
     → ともだちに聞いてみてください。
  4. 納豆(なっとう)・食べる
     → 納豆を食べてみます。
     → 納豆を食べてみてください。

練習２

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：イチゴのケーキ・作る
 → イチゴのケーキを作ってみます。
 → イチゴのケーキを作ってみましょう。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 宿題(しゅくだい)・やる
     → 宿題をやってみます。
     → 宿題をやってみましょう。
  2. 新(あたら)しいスキー場(じょう)・遊(あそ)びに行く
     → 新しいスキー場に遊びに行ってみます。
     → 新しいスキー場に遊びに行ってみましょう。
  3. 先生・質問(しつもん)する
     → 先生に質問してみます。
     → 先生に質問してみましょう。
  4. 天(てん)ぷら・揚(あ)げる
     → 天ぷらを揚げてみます。
     → 天ぷらを揚げてみましょう。
