練習問題

「〜してから〜する」

練習１

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：ごはんを食べる・風呂に入る
 → ごはんを食べてから風呂に入ります。
 → 風呂に入ってからごはんを食べます。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 焼(や)く・切(き)る
     → 焼いてから切ります。
     → 切ってから焼きます。
  2. テニスをする・プールで泳(およ)ぐ
     → テニスをしてからプールで泳ぎます。
     → プールで泳いでからテニスをします。
  3. 学校(がっこう)に行(い)く・朝食(ちょうしょく)をとる
     → 学校に行ってから朝食をとります。
     → 朝食をとってから学校に行きます。
  4. 映画(えいが)をみる・ショッピングをする
     → 映画をみてからショッピングをします。
     → ショッピングをしてから映画をみます。

練習２

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：宿題をする・寝(ね)る
 → 宿題をしてから寝なさい。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 免許(めんきょ)をとる・車に乗(の)る
     → 免許をとってから車に乗りなさい。
  2. 合格(ごうかく)する・バイトする
     → 合格してからバイトしなさい。
  3. 就職(しゅうしょく)する・結婚(けっこん)する
     → 就職してから結婚しなさい。

練習３

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：宿題をする・寝(ね)る
 → 寝るのは宿題をしてからにしなさい。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 免許(めんきょ)をとる・車に乗(の)る
     → 車に乗るのは免許をとってからにしなさい。
  2. 合格(ごうかく)する・バイトする
     → バイトするのは合格してからにしなさい。
  3. 就職(しゅうしょく)する・結婚(けっこん)する
     → 結婚するのは就職してからにしなさい。
