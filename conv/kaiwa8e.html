練習問題

「〜やすい」「〜にくい」

練習１

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：わかる・山田先生の授業
 → 山田先生の授業はわかりやすいです。
 → 山田先生の授業はわかりにくいです。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 食べる・焼(や)き魚(ざかな)
     → 焼き魚は食べやすいです。
     → 焼き魚は食べにくいです。
  2. 運転する・日本車(にほんしゃ)
     → 日本車は運転しやすいです。
     → 日本車は運転しにくいです。
  3. 飲む・このお酒
     → このお酒は飲みやすいです。
     → このお酒は飲みにくいです。
  4. 走る・台湾の高速道路(こうそくどうろ)
     → 台湾の高速道路は走りやすいです。
     → 台湾の高速道路は走りにくいです。

練習２

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：食べる・焼(や)き魚(ざかな)・骨(ほね)が多(おお)い
 → 焼き魚は骨が多くて食べにくいです。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 暮(く)らす・北国(きたぐに)・気温(きおん)が低(ひく)い
     → 北国は気温が低くて暮らしにくいです。
  2. 運転する・日本車(にほんしゃ)・性能(せいのう)が良い
     → 日本車は性能が良くて運転しやすいです。
  3. 飲む・このお酒・辛口(からくち)だ
     → このお酒は辛口で飲みやすいです。
  4. 走る・台湾の高速道路(こうそくどうろ)・車(くるま)が多(おお)い
     → 台湾の高速道路は車が多くて走りにくいです。
