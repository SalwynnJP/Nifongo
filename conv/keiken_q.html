練習問題

「〜したことがある」

練習１

 次(つぎ)の例(れい)のように文(ぶん)を書(か)きましょう。

例：行く・東京・私
 → 私は東京に行ったことがあります。
 → 私は東京に行ったことがありません。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. 焼(や)く・ケーキ・彼
     → 彼はケーキを焼いたことがあります。
     → 彼はケーキを焼いたことがありません。
  2. 折(お)る・骨(ほね)・山田くん
     → 山田くんは骨を折ったことがあります。
     → 山田くんは骨を折ったことがありません。
  3. 富士山(ふじさん)・登(のぼ)る・姉(あね)
     → 姉は富士山に登ったことがあります。
     → 姉は富士山に登ったことがありません。

練習２

 適当(てきとう)な言葉(ことば)を入(い)れましょう。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

    • すき焼きは食べたことがありますか？
    • いいえ。食べた ことはありません 。
    • ハワイに行ったことがありますか？
    • いいえ。ハワイは行った ことがありません 。
    • 『砂(すな)の女(おんな)』を 読んだことがありますか ？
    • はい。読んだことがあります。
    • いつ 読んだんですか ？
    • 高校生のときに読みました。
    • プロ野球を観戦(かんせん)しに行ったことはありますか？
    • ええ。 あります よ。
    • 誰(だれ)と行きましたか？
    • 友だちといっしょに 行きました 。

練習３

 適当(てきとう)な言葉(ことば)を入(い)れましょう。

 答えが表示されます。 ←の部分にポインタをのせると答えが出ます。

  1. ハワイには何回(なんかい)か 行ったことがあります 。
  2. すき焼(や)きはまだ 食べたことがありません 。
  3. 100点は一度(いちど)も とったことがありません 。
  4. 幽霊(ゆうれい)を２回くらい 見たことがあります 。
  5. 一度(いちど)だけ冒険(ぼうけん)を したことがあります 。
